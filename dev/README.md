# Development Tools

This directory contains development utilities for the Vibe Stats extension.

## Icon Generator

### `icon-generator/create-ai-vibe-icons.html`
The primary AI-themed vibe icon generator for the extension. This tool creates all four required icon sizes (16px, 32px, 48px, 128px) with the current AI-vibe aesthetic.

**Usage:**
1. Open the HTML file in your browser
2. Icons are auto-generated with download links
3. Download all four sizes (`ai-vibe-16.png` through `ai-vibe-128.png`)
4. Replace existing files in the `/icons/` folder
5. Reload extension in Chrome to see changes

**Features:**
- Modern AI-themed gradient design
- High-resolution optimized for modern displays
- Consistent with extension branding
- All required Chrome extension icon sizes
